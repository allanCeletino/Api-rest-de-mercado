# Api-rest-de-mercado
O Mercado API é um ecossistema backend robusto e de alta performance desenvolvido em Java 17 e Spring Boot 3.x, projetado para automatizar as operações comerciais de um mercado. Utilizando o padrão de arquitetura em camadas (Controller, Service, Repository e Model), o sistema garante total separação de responsabilidades. completo e funcionado tudo.
